Description of Tools:

This project was divided into two sections. The first part of the project consisted of creating a RESTful web server. We used this to populate our table with the necessary data. The second part of the project consisted of implemeting an interactive map using leaflet. This map allows users to see where each neighborhood is with markers depicting how many crimes are within a certain range. We also used the nominatim API to take user input and convert it to the corresponding address/latitude and longitude values. Vue was used as the client-side framework in this project.
